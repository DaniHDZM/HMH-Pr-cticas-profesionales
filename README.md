# HMH-Pr-cticas-profesionales
Prueba técnica de HMH

---



## Herramientas usadas

La prueba fue realizada con Vue + Vite.

Se utilizo Vue Router para la navegación.

Bootstrap para el formulario, tabla y navbar inciales tambien se incluyeron algunos SVG de Bootstrap.

Axios para el consumo de API

El objetivo principal fue demostrar habilidades en el desarrollo frontend, manejo de estado, enrutamiento y diseño responsivo.

---

## Instalación y Uso

git clone https://github.com/DaniHDZM/HMH-Pr-cticas-profesionales.git

cd HMH-Pr-cticas-profesionales

npm install

npm run dev
